# 订单与物流常见问题

Q：订单显示已发货，为什么还没有物流轨迹？

A：商家回传运单号后，承运商首次揽收可能存在延迟。发货后 24 小时仍无轨迹时可发起物流核查；具体订单状态需调用 `query_logistics_status`。

Q：物流显示已签收，但我没拿到商品怎么办？

A：先核对收货地址、签收人、代收点和承运商证明。仍无法找到时，创建“签收未收到”工单，不能仅凭用户口述直接退款。

Q：可以在发货后修改地址吗？

A：已出库订单不保证可以改址。可先查询实时物流状态，若承运商支持则转人工协调，客服 Agent 不直接修改物流信息。
