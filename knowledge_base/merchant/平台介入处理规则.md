# 平台介入处理规则

## 申请条件

用户与商家已就同一售后问题沟通但未达成一致，商家超过工单时限未处理，或涉及虚假履约、重大质量与安全风险时，可申请平台介入。平台介入是审核流程，不等于立即支持任一方。

## 材料整理

系统应整理订单快照、用户诉求、商家回复、双方凭证、物流轨迹和已执行动作。模型可以总结材料，但不能修改原始凭证或隐藏对一方不利的事实。

## 结果执行

平台结论可包含退款、退货、补发、补偿、驳回或继续举证。涉及资金或库存的执行应调用受控业务工具，使用审核结论 ID 做幂等和审计关联。
